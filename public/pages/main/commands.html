Terminal Commands

Deploying to Github

$ git init (to intiliaze your project - assuming you are already in your project folder)
$ git add . (this will stage all your files)
$ git commit -m "message"
$ git remote add origin 'repository url' (this is added your repository as remote-repo. Do it once)
$ git pull origin main (this will pull first before pushing as it is best practice)
$ git push (to push to the remote repo)
